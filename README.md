# E-commerce SQL Analysis

##  Project Overview
This project analyzes a real-world e-commerce dataset using SQL to extract meaningful business insights.



##  Skills Used
- SQL
- Joins
- Aggregations (SUM, COUNT)
- Data Cleaning
- Data Analysis



##  Key Insights
-  Total revenue: 16M+
-  Most orders are successfully delivered (~96%)
-  Sao Paulo is the top city by number of orders
-  High-value orders identified (up to 13k+)



##  Dataset
Brazilian E-commerce Dataset (Olist)

All queries used in this project are available in:

##  Conclusion
This project demonstrates how SQL can be used to analyze business data and generate actionable insights.
